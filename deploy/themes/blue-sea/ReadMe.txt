Istallazione
=============
Per istallare correttamente il tema neme-sys sul vostro sistema, eseguire i seguenti passaggi:

1) caricare sul vostro server, o sul vostro pc locale, nella directory designata per la vostra applicazione web, tutti i file estratti dallo zip di istallazione (sovrascrivendo tutti i file già presenti);
2) aprire il browser all'url: http://www.vostro_sito.com/public/install/themeinstall.aspx (se lavorate sul vostro pc locale, sostituire www.vostro_sito.com con localhost)
3) al termine dell'istallazione sarete inviati direttamente al BackOffice;
4) Nel BackOffice troverete caricati i nuovi template del tema;

l'utente amministratore di default è:
user: administrator
pwd: admin

Installations
=============
To install neme-sys theme on your system, perform the following steps:

1) load on your server, or on your local PC, under the directory designated for your web application, all files and directories extracted from the installation .zip (overriding every file already present);
2) Open the browser at the URL: http://www.your_site.com/public/install/themeinstall.aspx (if you work on your local computer, replace www.your_site.com with localhost)
3) After installation you will be sent directly to BackOffice;
4) inside the BackOffice you'll find every new template of thema already load;

default administrator user is:
user: administrator
pwd: admin
